类

1.类的声明与实例化

  • 使用class 类名进行声明类(可以不用先定义类)
  • 类的实例化可以使用 类名 实例名(构造函数) 或 类名 *实例名 = new 类名(构造函数)的方法进行实例化
  • 使用 类名() = default;可以初始化类

    2.成员函数,构造函数

  • 成员函数一般在类内声明在外部定义
  • 构造函数与类名相同可以使用实参(形参),的方式进行赋值
  • 内联函数inline通常适用于小型函数
  • vector 容器在使用类时，类需要有默认的构造函数
  • 友元函数friend，使用修饰词后可以直接调用私有参数

    3.this指针,静态资源,继承

  • 通常指向该类本身,可以使用return *this返回修改参数后的实例
  • 使用static修饰后的值不能在类中进行初始化，必须在类外使用类名::静态资源名进行初始化
  • 使用 类:父类1,父类2即可继承，其他规则同java

    4.重载,虚函数(接口)

  • 函数重载:相同函数名不同参数即可
  • 操作符重载定义 operator 操作符(){自定义函数}即可重载操作符 Distance ++ 为operator ++(int)
  • .：成员访问运算符

    .*, ->*：成员指针访问运算符

    ::：域运算符

    sizeof：长度运算符

    ?:：条件运算符

#： 预处理符号

这些无法重载

  • 使用vitual 返回值类型 函数名(形参) = 0;即可构造接口,可在子类中对接口进行实现(即同名同参函数);
